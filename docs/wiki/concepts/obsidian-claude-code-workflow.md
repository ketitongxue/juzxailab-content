# Obsidian 与 Claude Code 工作流

Obsidian 与 Claude Code 工作流是在 Markdown 知识库和 AI 编程工具之间搭建的轻量桥梁。原笔记提出安装 Claude Code 的 Obsidian 插件，并向当前项目加入面向 Obsidian 的技能。

## 模式

该工作流把 Obsidian 作为持久知识界面，把 [Claude Code](/wiki/entities/claude-code)作为执行界面。笔记保存上下文、指令或课程材料；Claude Code 则能把这些笔记摄取到结构化 wiki 或项目工作流中。

对该 wiki 而言，关键操作经验是完整捕获附件：摄取本地 Obsidian 笔记时必须同时归档图片嵌入，使原始笔记在迁移后仍可查看。
