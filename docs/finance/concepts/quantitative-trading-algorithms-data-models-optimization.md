# 量化交易：算法、分析、数据、模型和优化

`量化交易` 中文版来自 `Quantitative Trading: Algorithms, Analytics, Data, Models, Optimization`。前言指出，金融科技革命和大数据革命使量化投资策略、统计模型、知识表达、数据分析、算法和信息在金融行业中越来越重要；本书强调算法交易和量化投资策略开发与执行的跨学科方法。

## 知识结构

本书标题本身给出五个核心维度：

- **Algorithms**：交易规则、执行逻辑和优化算法。
- **Analytics**：统计分析、市场结构和策略诊断。
- **Data**：历史数据、实时数据、另类数据和数据质量。
- **Models**：金融统计、时间序列、风险模型和预测模型。
- **Optimization**：组合优化、交易执行优化和参数优化。

## 实践提醒

量化交易不是只写策略信号；数据、模型、执行和优化同等重要。大型量化系统的失败往往不是单一公式错误，而是数据延迟、成本假设、交易约束、模型过拟合和执行偏差共同作用。
