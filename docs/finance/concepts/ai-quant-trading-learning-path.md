# AI 量化交易学习路径

这份课程材料把 AI 量化交易定义为一个“AI 量化工作室”的构建过程：从金融与数据基础、回测与指标计算，到策略进阶、AI 投研、实盘执行、风控和可视化控制台。核心目标不是单个脚本，而是形成由 [ai-quant-agent-workflow](/finance/concepts/ai-quant-agent-workflow) 驱动、以 [backtesting-bias-and-frictions](/finance/concepts/backtesting-bias-and-frictions) 校验、通过 [qmt-miniqmt-trading-system](/finance/concepts/qmt-miniqmt-trading-system) 落地的交易闭环。

## 学习收益

- **个人财富管理**：用严格回测替代凭感觉交易，理解未来函数、幸存者偏差、滑点、手续费和交易限制。
- **职业能力**：对标金融机构 AI 应用落地工程师、量化开发工程师和量化策略研究员所需能力。
- **系统化能力**：不仅学习策略，还学习数据、回测、因子、AI 投研、风控、实盘接口和 Web 控制台。

## 课程阶段

1. **基建与招募**：确立投资哲学，学习市场基础，部署 Python/Anaconda/VSCode，接入 Tushare/Baostock/QMT/XtQuant。
2. **策略进阶**：海龟交易法则、缠论量化、网格、多因子、XGBoost/LightGBM 因子挖掘、论文复现与策略进化。
3. **AI 进化**：RAG 读取财报与公告，国泰君安“五步法”生成研报，舆情与事件驱动，XtQuant 实盘接口，强化学习执行探索。
4. **团队合体**：用 LangGraph 编排 Charles、Zoe、Kris、Ethan，完成盘前晨会、信号生成、风控审核、下单执行和复盘。
