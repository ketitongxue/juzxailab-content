# 缠论的量化拆解

缠论（缠中说禅）的形态学在 AI 量化视角下，本质是一套处理市场分形数据的**递归算法**。本页用信号处理与计算机数据结构的语言，把"一千个人眼里一千个缠论"统一为唯一可执行的信号。分形几何基础见 [fractal-market-structure](/finance/concepts/fractal-market-structure)。

## 1. 结构层级（递归数据结构）

| 层级 | 概念 | 量化翻译 |
|------|------|----------|
| 原子 | K线包含处理 | 非线性降维/平滑/池化层 |
| 细胞 | 分型（顶/底） | 分形最小转折单元 |
| 组织 | 笔 | 低通滤波器（空间+时间过滤） |
| 器官 | 线段（≥3笔+重叠） | 趋势确认（N字形验证） |
| 生命体 | 中枢（≥3线段重叠） | 价值密集成交区/引力中心/黑洞 |
| 宇宙 | 级别 | 1min→5min→30min→日线递归 |

### K线包含处理
将纠缠重叠的K线按"高高取高、低低取低"合并成一根标准K线，提取主特征、剔除噪音，类似 CNN 池化层。

### 分型
顶分型（^形）/底分型（V形）—— 分形中最小的转折单元。

### 笔
顶分型 + 底分型 + 中间≥1根独立K线构成一笔。**双重过滤**：空间（分型结构）+ 时间（独立K线），比只看幅度的 ZigZag 更严格，过滤掉瞬间的跳动。

### 线段
至少3笔（下-上-下 或 上-下-上）且有重叠构成线段。对应"前进—后退蓄力—再前进"的N字验证，胜率高于在笔级别交易。

### 中枢
连续3个次级别走势重叠的部分，物理意义为：

- **筹码密集区**：大量成交堆积；
- **价值吸附区/黑洞**：价格涨跌都会被拉回；
- **均值回归中心**：围绕波动的平衡点。

牛市 = 中枢不断上移；熊市 = 中枢不断下移；盘整 = 围绕同一中枢震荡。中枢是**结构性**指标（告诉你"现在在哪"），优于 MACD/KDJ 这类线性指标（只告诉你"昨天到今天涨没涨"）。

## 2. 背驰（动量衰竭）

只有在**趋势**（≥2个同级别中枢）中才存在背驰。判断公式：

```
Momentum(c) < Momentum(b)   # 离开第二中枢力度 < 离开第一中枢力度
```

### 量化力度方法
- **MACD 面积法**：b段 vs c段 MACD 红绿柱总面积积分对比，若 c 段面积缩小且价格创新低，即趋势背驰。
- **斜率法**：c段下跌斜率变平缓 → 空头力量衰竭。

### 趋势背驰 vs 盘整背驰
- **趋势背驰**：发生在两个中枢之后，**大反转信号**（胜率高，做）。
- **盘整背驰**：发生在中枢内部，往往只是小反弹（风险大，不做）。

## 3. 三类买卖点（盈亏比 vs 胜率）

| 买点 | 定义 | 本质 | 特征 |
|------|------|------|------|
| 1B | 趋势背驰引发的绝对底 | 左侧交易/均值回归/接飞刀 | 利润最厚但最危险，需多因子共振（VIX极致+估值分位） |
| 2B | 1B反弹后回撤不破前低 | 双底/次低点，结构确认 | **胜率最高**，盈亏比与胜率平衡最好 |
| 3B | 突破中枢后回踩不进中枢 | 趋势跟随/突破确认 | 短期爆发力最强，需放量突破+缩量回踩 |

1B 抄底、2B 建仓、3B 加仓——三类买点对应熊末牛初、牛市初期、主升浪的不同阶段。

## 4. 区间套（显微镜逐级放大）

**大级别的买点，一定对应小级别的买点。** 日线 1B 即将形成时，依次在 30分钟 → 5分钟 → 1分钟 镜头下找到 1分钟底分型确立后下单。这一刻同时买到了多周期的底——多周期共振的几何学表达。AI 可用递归函数毫秒级监控全市场，人类大脑很难同时处理4个周期。

### ⚠️ 信号闪烁问题
Tick 数据流中小级别底分型（特别是1分钟）极不稳定：这一秒成立，下一秒消失。**绝不能用当下最高/最低价触发（未来函数），必须用已完成的 Close 价确认。** 量化缠论不追求买在最低分钱，只追求**结构的确定性**——这与 [backtesting-bias-and-frictions](/finance/concepts/backtesting-bias-and-frictions) 的反未来函数原则一致。

## 5. 核心结论

- 中枢是骨架（位置）→ 背驰是肌肉（变盘信号）→ 三类买卖点是神经（行动）→ 区间套是瞄准镜（精确时刻）。
- 如果图形乱、无清晰中枢和背驰，AI 指令是 **Pass（放弃）**。
- 性格定位：求暴利研究 3B，求稳健死磕 2B，绝大多数人远离 1B。
