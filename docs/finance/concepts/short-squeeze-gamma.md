# 逼空与伽马挤压

2021 年，一群散户利用社交媒体集结，通过期权和现货联动，把华尔街顶级对冲基金 Melvin Capital 逼到破产边缘。GME 股价从 2 美元一路飙升到 483 美元。市场结构（期权对冲机制）比基本面更重要，情绪是可以被量化、被利用的因子。

## GME 逼空事件（2021）

游戏驿站（GameStop）是卖实体游戏光盘的连锁店，数字化冲击下看似注定倒闭。华尔街对冲基金像秃鹫一样做空。

**做空比例高达 140%**——市面上只有 100 股，却被卖空了 140 股（A 借给 B 卖空，B 卖给 C，C 又借给 D 卖空……）。这是一个堆满炸药桶的房间，只要一点火星就爆炸。

## 第一重武器：轧空（Short Squeeze）

带头大哥 Keith Gill（专业金融分析师）发现 GME 被严重低估且做空比例太高，在 Reddit 的 WallStreetBets（WSB）晒出多单，喊出 **YOLO（You Only Live Once）**。散户集结买入，轧空正反馈启动：

1. Melvin Capital 做空亏损。
2. 为止损/被追加保证金，必须买回股票还给券商。
3. 买回的动作推高股价。
4. 股价更高 → 亏损更多 → 需要买回更多 → 股价更高。

这是正反馈循环，但只是常规武器。真正让华尔街崩溃的是第二重核武器。

## 第二重武器：伽马挤压（Gamma Squeeze）

散户资金有限，于是疯狂买入**看涨期权（Call Options）**——几百美元就能控制价值几万美元的股票。

做市商（如 Citadel）卖期权给你，要保持中性。买一份 Call 时，做市商持有空头头寸，必须在股票市场买入正股对冲。买多少由 **Delta 值**决定。

- 股价低时期权虚值，Delta 小（如 0.1），做市商只买一点点。
- 股价上涨时期权变实值，Delta 迅速变大（接近 1）。
- **Gamma = ΔDelta / ΔPrice（加速度）。**

**死亡螺旋：**
1. 散户买入 Call → 做市商对冲买入正股。
2. 做市商买入 → 推高股价。
3. 股价上涨 → Gamma 效应 → Delta 变大。
4. Delta 变大 → 做市商必须再买更多正股。
5. 做市商买入更多 → 股价继续暴涨……

> **做市商变成了散户的免费打手。** 散户每买一张期权，就强迫做市商像机器一样买正股。这是物理学上的共振——散户找到了系统固有频率，把这座大桥震塌了。

## 情绪因子：钻石手 = 流动性锁

传统模型假设投资者理性（赚了钱会止盈）。Melvin Capital 以为散户会卖，扛一扛就过去。但遇到了不理性的疯子——**钻石手（Diamond Hands）**文化：哪怕亏光也不卖，谁卖谁是叛徒。

量化视角解读——**流动性枯竭**：
- 买盘：源源不断的做市商回补 + 空头止损。
- 卖盘：几乎为零（散户锁仓）。

供需曲线垂直时，价格由买方说了算。Melvin Capital 想平仓？没人卖给你。最终巨亏 53%，寻求注资救命。

> 传统基本面量化失效——看 PE、PB、现金流全是错的。**唯一的因子就是情绪。**

## 拔网线：游戏终结

2021 年 1 月 28 日，散户交易软件 Robinhood 宣布**禁止买入 GME，只能卖出**（因清算所 DTCC 要求缴纳巨额保证金，Robinhood 没钱了）。拔网线切断买盘，供需瞬间逆转，Gamma Squeeze 反向运行，股价从 483 美元自由落体。

## 舆情监控（AI 新课题）

GME 事件后，量化基金意识到必须监控社交媒体：

| 指标 | 方法 |
|------|------|
| 提及量 | GME 今天被提到多少次？比昨天暴增 1000% = 预警 |
| 情绪极性 | 骂还是夸？ |
| Emoji 分析 | 🚀（火箭）和 💎🙌（钻石手）密集出现 = 极度狂热/逼空前兆 |

结合期权异动：舆情热度飙升 + 看涨/看跌比异常 + 隐含波动率（IV）暴涨 = **伽马挤压可能正在形成**。

## 铁律

> **永远不要做空波动率无限的资产（Meme Stocks）。**

- 做多最大亏损 100%（归零）。
- 做空最大亏损**无穷大**（价格可以涨到天上去）。

AI 提示某股票具有"Meme 属性"时，哪怕基本面再烂，也禁止做空。

## 实战建议

1. **关注结构性拥挤**：做空比例超过 30% 的股票，空头太拥挤，一点利好就踩踏式上涨——反而是做多机会（博弈空头回补）。
2. **情绪是短期最强因子**：A股的连板战法。当情绪被点燃时估值没有意义。用 AI 监控情绪热度：满屏火箭时跟随买入，火箭数量下降时果断离场，不要做最后的钻石手。
3. **资金优势不是绝对的，规则优势才是**：散户利用金融工程规则（期权对冲机制），蚂蚁咬死大象。
