# 凯利公式 (Kelly Criterion)

## 概述

凯利公式（Kelly Formula / Kelly Criterion）是量化交易中**仓位管理的基石**。它回答一个核心问题：当你拥有概率优势时，应该投入多少资金？[索普](/finance/entities/edward-thorp)将其引入赌博和金融交易，使其成为后来量化交易仓位管理的"圣经"。

## 核心原理

凯利公式源自 John L. Kelly 1956年的论文 *A New Interpretation of Information Rate*（《信息率的新解释》）。

### 香农—凯利—贝尔实验室：信息论到赌博的跨界

1956年，信息论之父**克劳德·香农**（Claude Shannon）正在贝尔实验室研究长途电话的信号噪音问题。他的同事，物理学家**约翰·凯利**（John L. Kelly），对同样的数学结构感兴趣——但他的动机不是电话线，而是**赌马**。

凯利的核心洞见：**电话线里的噪音和赌局里的内幕消息在数学上是一回事。** 如果你有信息优势（知道某匹马的真实胜率高于赔率隐含的概率），你的财富增长速率与信号传输中的信息率遵循相同的数学规律。这就是为什么这篇论文发表在信息论领域，而非赌博文献中——凯利公式本质上是**信息论的金融应用**。

**基本形式（二元结果）：**

$$f^* = \frac{bp - q}{b} = \frac{p(b+1) - 1}{b}$$

| 符号 | 含义 |
|------|------|
| $f^*$ | 最优下注比例（占总资金） |
| $b$ | 赔率（赢时的净收益/亏时的损失） |
| $p$ | 赢的概率 |
| $q$ | 输的概率 ($q = 1 - p$) |

**直觉理解：**
- 优势越大（$p$ 越高），下注比例越大
- 赔率越好（$b$ 越大），下注比例越大
- 没有优势（$p \leq 1/(b+1)$）时，$f^* \leq 0$ → 不下注

## 索普的应用

[索普](/finance/entities/edward-thorp)将凯利公式应用于21点算牌：

> "优势大时（大牌多），下重注；优势小时（小牌多），下最小注甚至不下注。"

这不是凭感觉，而是根据当前牌堆的胜率变化，**动态计算最优下注比例**。这正是 [正期望值策略](/finance/concepts/positive-expected-value) 不被波动清洗出局的关键。

## 为什么不全仓梭哈？

当优势只有1%时，全仓梭哈会在波动中爆仓——即使长期期望值为正，短期的连续亏损也可能让你归零（破产风险）。

凯利公式的核心洞见：
1. **最大化长期复利增长率**（而非单次期望收益）
2. **在足够大的样本量下，资金永远不会归零**（$f^*$ 保证了生存）
3. **动态调整** — 优势变化时，仓位也随之变化

## 索罗斯式非对称机会：为什么下一讲回到凯利

第4讲的 [索罗斯](/finance/entities/george-soros) 强调：胜率不重要，赔率和仓位才重要。他的英镑战役是典型的 [非对称赔率](/finance/concepts/asymmetric-payoff-and-position-sizing)：错了小亏，对了暴赚。

这正好引出凯利公式的必要性：找到非对称机会只是第一步，真正困难的是**下注多少**。下注太小，抓不住大机会；下注太大，即使判断方向正确，也可能在路径波动中先破产。

## 第5讲深化：波动率税与半凯利

### 算术平均 vs 几何平均（核心陷阱）

一个60%胜率的硬币游戏（赔率1:1），凯利最优仓位 f\* = 20%。但如果每次下注50%：

$$1.5^{60} \times 0.5^{40} \approx 0.00003$$

**即使拥有60%胜率，仓位过重（50%）最终也会输光。**

在复利世界里，亏损50%需要涨100%才能回本。下跌对资产的杀伤力远大于上涨。这就是**波动率税（Volatility Tax）**。

### 凯利曲线的悬崖效应

| 仓位区域 | 效果 |
|---------|------|
| 最优仓位左边（< f\*） | 少赚，但平缓安全 |
| 最优仓位右边（> f\*） | 多赚短期，长期收益率陡降甚至归零 |

### 半凯利（Half-Kelly）实践

> "宁可少赚，绝不爆仓。"

- 凯利说买20%，实际买10%
- 凯利说满仓（100%），实际买50%
- 量化大师（包括 [索普](/finance/entities/edward-thorp)本人）都提倡半凯利甚至1/4凯利

原因：真实市场中你**永远不知道精确的胜率**。回测出来的60%可能实际只有51%。过度自信 → 实际过度下注 → 悬崖边坠落。

### 动态仓位五步法

1. AI模型实时预测当前胜率 p（根据情绪、波动率动态变化）
2. 根据止盈止损位计算盈亏比 b
3. 代入凯利公式算出理论仓位 f\*
4. 乘以安全系数（0.5 或 0.3）
5. 波动率（VIX）过大时进一步降低仓位

> 胜兵先胜而后求战，败兵先战而后求胜。凯利公式就是那个让你先胜的数学保障。

## 实践中的考量

| 问题 | 实践解法 |
|------|---------|
| **过度下注风险** | 实践中常用"半凯利"（Half-Kelly）：下注 $f^*/2$，牺牲少量增长率换取大幅降低波动 |
| **参数不确定** | 真实交易中 $p$ 和 $b$ 难以精确估计；过度自信 → 实际过度下注 → 爆仓 |
| **多资产组合** | 需扩展为多变量凯利公式，考虑资产间相关性 |
| **离散vs连续** | 金融交易接近连续，需用连续版本 |

## 与风险管理的关联

凯利公式天然连接了 [positive-expected-value](/finance/concepts/positive-expected-value)（找到优势）和风险控制（不被清洗出局）：

```
正期望值策略 → 凯利公式计算仓位 → 大数定律兑现优势
     ↑                                      ↓
     └────── 回测验证（[backtesting-bias-and-frictions](/finance/concepts/backtesting-bias-and-frictions)）──────┘
```

它也与 [quant-performance-attribution](/finance/concepts/quant-performance-attribution) 中的最大回撤分析直接相关——过度下注会导致回撤超出承受范围。

## 常见误区

- ❌ "有优势就该满仓" — 凯利公式明确否定了这一点，$f^*$ 几乎永远 < 100%
- ❌ "凯利公式保证每次都赢" — 不，它只保证长期增长率最优，短期仍可能大幅亏损
- ❌ "精确估计胜率和赔率很容易" — 实践中参数估计是最难的部分，建议用 Half-Kelly
