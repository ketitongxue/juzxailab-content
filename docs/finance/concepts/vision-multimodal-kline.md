# 视觉与多模态 K 线分析

传统量化只能读取 OHLC 四个数字，对 K 线形态"盲人摸象"。计算机视觉与多模态大模型让 AI 像人眼一样识别图形，把"盘感"从玄学变成可工程化的能力。这是 [ai-quant-agent-workflow](/finance/concepts/ai-quant-agent-workflow) 的关键感知层。

## 1. 传统量化的局限：盲人摸象

`if Close < MA20 and Open > MA20` 这行代码只判断跌破均线，却不知道这根 K 线是：
- 带长下影线（探底回升）？还是
- 光头光脚大阴线（空头屠杀）？

为描述形状，程序员要写几百行代码定义"上影线 > 实体2倍"，既累又僵化。OHLC 四数字丢失了人类视觉可秒辨的形态信息。

## 2. CNN 计算机视觉：像素级洞察

卷积神经网络处理 K 线图与处理猫狗照片的方式一致——不看数字，看像素。

| 组件 | 作用 |
|------|------|
| 卷积 Convolution | 像放大镜滑动，提取横线/竖线/拐角特征 |
| 池化 Pooling | 压缩特征，忽略细节保留骨架 |
| 全连接 | 拼接骨架判断是猫、狗、还是"头肩顶" |

**模糊识别能力 = 容错率**：喂 1 万张 W 底图片训练后，模型能在第 10001 张图中精准圈出哪怕稍微歪一点、多一根杂乱 K 线的 W 底。这解决了传统量化过于僵化的痛点。

## 3. YOLO 目标检测识别缠论中枢

工业界最著名的目标检测算法 YOLO（You Only Look Once），常用于自动驾驶识别行人。量化实战中用它识别交易形态。

### 训练缠论猎手的流程
1. **标注**：懂 [changlun-quantitative](/finance/concepts/changlun-quantitative) 的老手在几千张 K 线图上框出中枢、三买点；
2. **训练**：把标注好的图喂给 YOLO；
3. **推理**：把实盘 K 线截图扔给模型。

### 输出示例
```
Box: Central Pivot,    Confidence: 0.95
Box: 3rd Buy Point,    Confidence: 0.88
```

不仅解决了"千人千缠"的标准化问题，更能捕捉**非标准但神似**的机会——某些走势按严格定义因差1分钱不算中枢破坏，但视觉上已明显破位，AI 能像老手一样忽略噪音直接发出卖出信号。

## 4. 多模态大模型（Qwen 实战案例）

若 CNN 只是"眼睛"，多模态大模型（Qwen-Max、Claude 4.5 等）就是"眼睛 + 大脑"——能看图、能 OCR 读数、能调工具、能做金融逻辑推理。

### 贵州茅台（600519）周线图实验
仅投喂一张图（含均线/MACD/RSI），不给任何数字，Qwen3-Max 表现惊艳：

1. **自主拆解**：不急于下结论，先观察再放大特定区域；
2. **工具使用**：自主调用 `image_zoom_in_tool` 聚焦 2025年12月低点；
3. **OCR 精准读数**：直接读出关键支撑位 1332.47；
4. **指标逻辑冲突推理**：
   - 均线 MA5 < MA10 < MA20 = 空头排列（趋势看跌）；
   - 但 MACD 柱状图由绿转红 = 下跌动能减弱、底背离雏形（反弹迹象）；
   - 低点成交量萎缩 = 观望情绪浓厚；
5. **结论像成熟投顾**：短线关注 1332.47 支撑、放量突破 MA5 轻仓；中长线等待均线多头排列更明确反转信号。

这是通用 AI（未专门训练茅台）仅凭一张图提取支撑位、趋势、动能、量价全维信息并完成逻辑闭环——**对传统量化策略的降维打击**。

## 5. 双脑协同：半人马交易系统

现阶段多模态大模型有两个短板：**慢**（分析一张图数秒，做不了高频）、**幻觉**（偶尔看错数字、把 RSI 看成 CCI）。解决方案是分工：

### 左脑（Python/C++）：负责筛选
- 硬规则代码快速扫描全市场 5000 只股票；
- 条件如：均线多头排列 + 换手率 > 5%；
- 结果：100 只候选股。

### 右脑（多模态 AI）：负责复核
- 批量发送 100 只 K 线截图给大模型；
- Prompt："请像资深技术分析师，看这10张图，哪些形态最像'空中加油'？哪些有顶背离风险？打分。"
- 结果：剔除 80 只形态杂乱，留 20 只精锐。

### 决策（人类）：负责开枪
只看 20 只股票，**工作效率提升 250 倍**。

## 6. 实战建议

1. **AI 作为第二双眼睛**：买入前截图发 Qwen，问"有哪些我看不到的风险？"——人类有"由爱生信"的偏见（屁股决定脑袋），AI 冷血指出 MACD 死叉+放量出货等被选择性忽略的危险信号，能省下大量冲动交易的钱。
2. **关注形态因子**：除收盘价单一数据外，引入视觉因子——K 线斜率、图形紧凑度、波动平滑度，这些曾无法量化的特征现在都能通过 CNN 提取成因子，纳入 [ai-quant-agent-workflow](/finance/concepts/ai-quant-agent-workflow) 的因子库。
3. **别迷信，要验证**：AI 视觉现阶段用于辅助决策与初筛，**绝对不能直接连实盘账户全自动交易**（除非有极强容错代码）。**AI 负责"看"，你负责"信不信"。**
